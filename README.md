# Paper Project Page

This repository is a static GitHub Pages template for a computer science paper website.

## Edit Checklist

1. Replace the paper title, venue, authors, affiliations, and one-line summary in `index.html`.
2. Update the resource buttons for the PDF, arXiv, code, and demo links.
3. Replace `assets/teaser-placeholder.svg` and `assets/result-placeholder.svg` with paper figures.
4. Paste the abstract, method overview, results captions, media embed, and BibTeX.
5. Update the page metadata in the `<head>` tag before publishing.

## Publish on GitHub Pages

Use the repository root as the Pages source. No build step is required.
